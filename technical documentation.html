Introduction

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It has imperative, object-oriented and generic programming features, while also providing facilities for low-level memory manipulation.

C++ is one of the most popular programming languages and is often used in systems programming, game development, and high-performance applications.

#include <iostream> int main() { std::cout << "Hello, World!"; return 0; }

Key features of C++:

  • High performance
  • Object-oriented programming support
  • Template metaprogramming
  • Direct memory access
  • Rich standard library (STL)
Basic Syntax

A C++ program consists of various entities such as variables, functions, classes, etc. The basic syntax includes:

// Single-line comment /* Multi-line comment */ #include <iostream> // Header file inclusion using namespace std; // Using standard namespace int main() { // Main function cout << "Hello"; // Statement return 0; // Return statement }

Basic components:

  • Semicolons - Terminate statements
  • Blocks - Group statements between { }
  • Identifiers - Names for variables, functions, etc.
  • Keywords - Reserved words with special meaning

Note: C++ is case-sensitive. "myVar" and "myvar" are different identifiers.

Data Types

C++ supports various data types that can be classified as:

int age = 30; // Integer float price = 19.99; // Floating point double pi = 3.1415926535; // Double precision char grade = 'A'; // Character bool isCppFun = true; // Boolean string name = "John Doe"; // String

Fundamental data types in C++:

  • Integer types: int, short, long, long long
  • Floating-point types: float, double, long double
  • Character types: char, wchar_t, char16_t, char32_t
  • Boolean type: bool
  • Void type: void

C++ also supports derived types like arrays, pointers, references, and user-defined types.

Control Structures

Control structures allow you to control the flow of execution in a program. The main control structures in C++ are:

Conditional statements:

if (x > 0) { cout << "Positive"; } else if (x < 0) { cout << "Negative"; } else { cout << "Zero"; } switch(grade) { case 'A': cout << "Excellent"; break; case 'B': cout << "Good"; break; default: cout << "Invalid"; break; }

Loops:

for (int i = 0; i < 10; i++) { cout << i << " "; } while (x > 0) { cout << x-- << " "; } do { cout << "This runs at least once"; } while (false);
Functions

Functions allow you to group code into reusable units. A function declaration includes:

// Function declaration int add(int a, int b); // Function definition int add(int a, int b) { return a + b; } // Function call int result = add(5, 3);

Key concepts about functions:

  • Parameters - Values passed to the function
  • Return type - The type of value the function returns
  • Function overloading - Multiple functions with same name but different parameters
  • Recursion - Functions that call themselves

Note: The main() function is special - it's the entry point of every C++ program.

Object Oriented Programming

C++ supports object-oriented programming with these key features:

class Car { // Class declaration private: // Access specifier string brand; // Attribute public: void setBrand(string b) { // Method brand = b; } string getBrand() { return brand; } }; int main() { Car myCar; // Create object myCar.setBrand("Toyota"); // Call method cout << myCar.getBrand(); return 0; }

Four pillars of OOP in C++:

  • Encapsulation - Bundling data and methods that work on that data
  • Abstraction - Hiding complex details and showing only essentials
  • Inheritance - Creating new classes from existing ones
  • Polymorphism - Ability to process objects differently based on their data type

C++ also supports advanced OOP concepts like operator overloading, friend functions, and virtual functions.
